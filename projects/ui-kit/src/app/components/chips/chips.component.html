Chip Components

Chip components for displaying tags, selections, and other compact information. Similar to PrimeNG's p-chip and p-chips components.

Basic Chips

Chips Input (Tags)

Required Validation

At least 2 tags are required

Features

  • Basic Chips: Display static information as compact chips
  • Removable Chips: Chips with remove functionality
  • Icon Support: Chips can display icons (using PrimeIcons)
  • Chips Input: Interactive input for creating multiple chips/tags
  • Reactive Forms: Full integration with Angular reactive forms
  • Float Labels: PrimeNG-style float label support
  • Validation: Works with Angular form validators
  • Keyboard Support: Enter, Tab, and Backspace key handling
  • Paste Support: Parse multiple values from clipboard
